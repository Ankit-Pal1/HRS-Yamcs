Configuration
=============

Example
-------

Web options are configured in the file ``etc/yamcs.yaml``.

.. code-block:: yaml

   yamcs-web:
     tag: Example Mission
     siteLinks:
       - label: Wiki
         url: https://example.com/wiki
         external: true
